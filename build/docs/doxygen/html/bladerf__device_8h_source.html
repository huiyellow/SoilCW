92  * The function also enable biastee of RX channel for receiving radar echos if enable_biastee is set during class construction

95  * @param ref_rx_config RX channel configurations for receiving reference signals. Set it to nullptr if no reference RX is required

103  /**
104  * Configure and enable the TX channels of the bladeRF 2.0 device.
105  * Configure the frequency, gain, bandwidth and sampling rate of the specified TX channels
106  * The TX channels are configured to use SC16 Q11 samples *with* metadata
107  * The biastee of the TX channel for transmitting radar pulses is enabled if enable_biastee is set during class construction
108  * @note User can enable only one TX channel to transmit radar pulse by setting ref_tx_config to nullptr
109  *
110  * @param ref_tx_config TX channel configurations for transmitting reference signals. Set it to nullptr if no reference TX is required
111  * @param radar_tx_config TX channel configurations for transmitting radar pulses.
112  * @param buf_config USB buffer configurations
113  * @param enable_biastee Whether biastees of TX (BT-100 power amplifier) and RX (BT-200 LNA) are connected.
114  * @return 0 if success
115  * */
116  int enable_tx_channels(struct channel_config *ref_tx_config, struct channel_config *radar_tx_config, struct usb_buffer_config *buf_config, bool enable_biastee);

118  /**
119  * Configure and enable TX and RX channels. It essentially call @see enable_rx_channels(), and @see enable_tx_channels() sequentially.
120  * ref_tx_config must be provided when using two TX channels, otherwise set ref_tx_config as nullptr (only enable radar TX channel)
121  * ref_rx_config must be provided when using two RX channels, otherwise set ref_rx_config as nullptr (only enable radar RX channel)
122  * @param radar_tx_config TX channel configurations for transmitting radar pulses
123  * @param radar_rx_config RX channel configurations for receiving radar echos
124  * @param ref_tx_config TX channel configurations for transmitting reference signals. Set it to nullptr if no reference tx is needed
125  * @param ref_rx_config RX channel configurations for receiving reference signals. Set it to nullptr if no reference rx is needed
126  * @param buf_config USB buffer configurations

128  * @return 0 if success.
129  * */
